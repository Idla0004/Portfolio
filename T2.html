LØSNING:

Temaet introducerede os til basale webteknologier som HTML og CSS. Vi opsatte vores første domæne og anvendte Visual Studio Code som primært udviklingsmiljø. Projektet tog udgangspunkt i fastlagt indhold og layout, hvor vi lærte at implementere mobile-first løsninger og anvende CSS Grid til strukturering af layoutet. Derudover fik vi indsigt i semantisk HTML og best practices inden for styling.

PROCESS

Selvom jeg tidligere havde stiftet bekendtskab med HTML og CSS, var det første gang jeg arbejdede ud fra et struktureret layoutdiagram. Det gav mig en ny forståelse for den visuelle planlægningsfase før kodning. Arbejdet blev udført i sparring med studiegruppen, hvilket understøttede fejlretning og optimering.

UDFORDRINGER I TEMA 2

En konkret teknisk udfordring opstod i opbygningen af galleri-layoutet på desktop, hvor ét billede ikke tilpassede sig korrekt i grid-strukturen. Problemet blev løst ved anvendelse af object-fit: cover.

HVAD JEG TOG MED FRA TEMA 2

Jeg tilegnede mig en stærkere forståelse for opbygning af layouts med CSS Grid og vigtigheden af at tænke i mobile-first løsninger. Dele som wireframes og layoutdiagrammer er siden blevet faste elementer i mit workflow, hvilket har øget min effektivitet i fremtidige projekter.
